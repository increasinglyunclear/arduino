autoplay_sequences.h

Click Download to save autoplay_sequences.h, then drop a copy into both mauwb_emitter/ and mauwb_receiver/ (alongside each .ino). Set AUTOPLAY_SEQUENCE near the top of each .ino to pick which sequence runs on boot — 1 = first in the list at the top of the file — then reflash.

Import autoplay_sequences.h

Paste the contents of autoplay_sequences.h below, or click Choose file… to load it from disk. By default this merges the imported sequences with whatever's already in the library (overwriting same-named entries). Tick Replace to wipe the existing library first.
